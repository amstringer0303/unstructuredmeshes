# unstructuredmeshes
# Introduction to Unstructured Meshes for Climate Change Analysis

This project provides an introduction to using unstructured meshes for analyzing climate change data. The project covers data collection, unstructured mesh generation, climate data analysis, and visualization.

## Project Structure

- `data/`: Contains raw and processed data.
- `notebooks/`: Jupyter Notebooks for each step of the project.
- `scripts/`: Python scripts for data collection, mesh generation, data analysis, and visualization.
- `README.md`: Project overview and instructions.
- `requirements.txt`: List of required Python libraries.
